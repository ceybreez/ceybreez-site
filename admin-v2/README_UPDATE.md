# CeyBreez Admin V10 Modern V2

This package keeps the stable V10 admin JavaScript/API workflow and applies an Admin V2 style sidebar/dashboard layout.

Upload the `admin-v2` folder to your GitHub repository root.

URL after deploy:

`https://ceybreez.com/admin-v2/admin.html`

Safe notes:
- Does not replace `/admin/` V10 live folder.
- Uses the same V10 `admin.js` logic and existing Worker API.
- No new D1 migration required.
- Manual booking/availability experimental Admin V2 modular code is not included in this rollback package.
